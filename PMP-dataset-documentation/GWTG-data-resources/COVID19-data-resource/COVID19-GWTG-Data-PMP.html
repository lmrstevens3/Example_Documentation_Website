About the AHA COVID19 CVD Registry Data

The COVID19 CVD Registry is a voluntary quality improvement registry, powered by AHA's Get With the Guidelines®.

This data was collected from x hospitals and comprises x patient records Upon collection, hospitals are provided detailed data abstraction instructions. In-form and post collection data quality checks are performed to ensure accurate data entry, flagging and preventing records from saving when a value is not within expected range.

The registry entry criteria instruct hospitals to enter all patients >=18 years old who are hospitalized with an active COVID-19 infection.Hospitals are directed to start data entry with their first hospitalized patient with a COVID-19 diagnosis and continue with abstraction in consecutive discharge order until current.

Additional information and documentation regarding the registry data, quality checks, data usage, as well as guidelines for analysis and publication are provided in the workspace as well as on this site.

Root Directory and COVID-19 CVD Data Location

  • The COVID-19 Registry Data is located in /mnt/workspace/GWTG/COVID-19
  • The root path in the workspace is /mnt/workspace. This is the default directory for the Jupyter Home Page JupyterLab, R Studio and SAS Studio.

Software

  • The workspace is fully customizable and tools such as SAS and MATLAB are available.
  • R Studio, Jupyter Notebook, JupyterLab, and SAS Studio (if requested) are available as tabs when a workspace is launched. The terminal and a variety of languages including R, Python, bash, and SAS (license required) can be accessed through Jupyter and JupyterLab.
  • A comprehensive list of pre-installed packages and software available can be found on the About Workspaces page.
