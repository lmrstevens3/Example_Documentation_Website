Introduction and Software¶

This notebook walks through loading, viewing and summarizing a GWTG dataset comprised of a sas7bdat data file and a sas7bcat formats file. The Tables and Summary statistics section provides a brief introduction for how to calculate frequencies and distribution statistics for categorical and continuous variables respectively.

Software¶

This tutorial is completed in SAS. All packages and libraries used in this tutorial are listed and installed in the sections below. Packages used in this tutorial as well as additional packages that are useful for tabulating and summarizing data are listed in the References Section.

Using SAS Studio¶

  • As shown below, the username and password for R Studio is rstudio and workspace respectively. Passwords and log in information are provided upon starting a workspace as well as available in the ReadMe.ipynb in /mnt/workspace.
  • This tutorial is accompanied with a SAS_covid19_cvd_data_usage.SAS File which can be opened and used in SAS Studio.

Creating Libraries¶

Creating a Library with the Library Window in SAS Studio¶

The new library button in the Libraries section of the navigation pane (left-side) in SAS Studio can be used to create Libraries. Once clicked, select the directory folder for the library.

Load and View Data¶

Importing files in SAS¶

The root directory path in the workspace is /mnt/workspace, and is the directory for the files listed under the Jupyter Home tab as well as the default working directory for Jupyter, JupyterLab, R Studio,and SAS Studio.

AHA COVID-19 CVD Registry Powered by Get With the Guidelines Data

The data, documentation, and usage examples are available in the following workspace folders:

  • Data: /mnt/workspace/GWTG/COVID19/data (includes data and format sas files)
  • Documentation: /mnt/workspace/GWTG/COVID19/documentation.

For a more indepth introduction to the data, check the COVID-19 CVD Registry Researcher Resource. The Explore and Discover section of this website can be used to review all documentation and explore variables available. The website is bookmarked in the favorites bar above.

In SAS Studio, Data can be imported by clicking the New button (far left button) under Server Files and Folders or selecting Import Data under Tasks and Utilities > Utilities.
Using SAS Code, The following command is used to load datasets in SAS, where we set the path for the data. We use the library name GWTG in this case followed by a period(.) and then the SAS filename that you need to access the contents.

  • PROC IMPORT can be used to load other files types, such as csv, if neeeded.

DATA COVID_COPY;

Summary Statistics¶

The PROC MEANS computes the summary statistics for the dataset. This method can be used to compute the summary statistics for the continuous variables in the dataset.

In [6]:
PROC MEANS DATA= GWTG.covid19_cvd_data_jul20 MEAN MEDIAN STD MIN MAX Q1 Q3 NMISS MAXDEC=2 ; 
    VAR AGEI; /*This can be used for any continuous variable from the dataset. */
