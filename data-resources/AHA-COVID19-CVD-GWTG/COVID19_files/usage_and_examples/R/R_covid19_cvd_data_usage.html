Introduction and Software¶

This notebook walks through loading, viewing and summarizing a GWTG dataset comprised of a sas7bdat data file and a sas7bcat formats file. The Tables and Summary statistics section provides a brief introduction for how to calculate frequencies and distribution statistics for categorical and continuous variables respectively.

Software¶

This tutorial is completed using R and provides R studio examples. All packages and libraries used in this tutorial are listed and installed in the sections below. Packages used in this tutorial as well as additional packages that are useful for tabulating and summarizing data are listed in the References Section.

Viewing the data¶

The following commands are helpful when viewing large datasets.

  • head() will show the first 6 rows (or the number of rows is passed to n). Similarly tail() will display the last 6 rows.
  • colnames() can be used to view the variable names (or field names) of the data.
  • str() will give an overview of the data type, label (description), and coding values.
    • To view all metadata for a given variable, the attributes() command can be used (ex. attributes(df$METHDIAG)).

NOTE: The colnames() and str() commands can be used on the entire dataset as well as a subset. Given the large size of the data, a subset used to limit the output displayed in the code cells below.

Tables and Summary Statistics¶

Tables and Frequencies¶

Getting the counts of one or multiple variables is easy in R with the table function. The table() function will create a set of counts. This function can be used to get counts of categorical variables in the data, as shown in the code below, which makes a table of the number of patients by the categories in the variable specified.

  • Tables can also be made for multiple variables. For example:
    • table(df$SEX, df$METHDIAG, useNA = "ifany")
  • The prop.table() function can be used to get a percentage of data for categorical variables. The round() function can be used to show the percentage with only 2 decimal places. For example:
    • round(prop.table(var_table)*100,2)
In [10]:
table(df$METHDIAG, useNA = "ifany")

   1    2    3 <NA> 
8200  666    3   51
In [11]:
#create table
var_table <- table(df$METHDIAG, useNA = "ifany") #useNA shows missing values if present
names(var_table) <- c("RT-PCR Test", "Clinical Diagnosis using Hospital Specific Criteria", "IgM antibody test", "Missing") #change rownames -1 means RT-PCR Test, 2 means Clinical Diagnosis using Hospital Specific Criteria, 3 means IgM antibody test

Summary Statistics¶

The summary() command can be used to view distribution statistics for all variables in the data or for a single variable. The summary command will display frequencies of categories for categorical variables that are read in as factors or converted to factors.

NOTE: summary() can be used on the entire dataset. Given the large size of the data, the command is called on a single variable below to limit the output displayed.*
