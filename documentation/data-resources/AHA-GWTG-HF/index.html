Root Directory and Heart Failure Data Location

  • The Heart Failure Registry Data is located in /mnt/workspace/GWTG/HF
  • The root path in the workspace is /mnt/workspace. This is the default directory for the Jupyter Home Page JupyterLab, R Studio and SAS Studio.

Software

  • The Transfer Data Section of the PMP User-Guide provides information regarding uploading files to the workspace or getting files out of the workspace.
  • The workspace is fully customizable and tools such as MATLAB are available upon request. R Studio, Jupyter Notebook, JupyterLab, and SAS Studio are available as tabs when a workspace is launched.
  • The terminal and a variety of languages including R, Python, bash, and SAS can be accessed through Jupyter and JupyterLab. A comprehensive list of pre-installed packages and software can be found on the About Workspaces page.
