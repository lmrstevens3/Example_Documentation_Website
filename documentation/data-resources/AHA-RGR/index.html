About the ® Research Goes Red Data

Root Directory and Data Location

  • The Research Goes Red Data is located in /mnt/workspace/Research_Goes_Red/
  • The root path in the workspace is /mnt/workspace. This is the default directory for the Jupyter Home Page JupyterLab, R Studio and SAS Studio.

Software

  • The Transfer Data Section of the PMP User-Guide provides information regarding uploading files to the workspace or getting files out of the workspace.
  • The terminal and a variety of languages including R, Python, bash, and SAS can be accessed through Jupyter and JupyterLab. A comprehensive list of pre-installed packages and software can be found on the About Workspaces page. The workspace is fully customizable and tools such as MATLAB are available upon request.
